# Sobre o projeto

Buscador de CEP é uma aplicação full stack web e mobile desenvolvida durante o Curso de Desenvolvimento de Software **Cubos Academy**

A aplicação consiste em uma pesquisa de CEPs, onde ao informar o CEP é retornado todas as informações do mesmo. Projto simples com intuito de aumentar meu conhecimento nas tecnologias utilizadas.

# Como executar o projeto

## Back end
Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://github.com/devsuperior/sds1-wmazoni

# entrar na pasta do projeto back end
cd backend

# executar o projeto
./mvnw spring-boot:run
```

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/devsuperior/sds1-wmazoni

# entrar na pasta do projeto front end web
cd front-web

# instalar dependências
yarn install

# executar o projeto
yarn start
```
